## Method

함수.

리턴 타입에 따라서 void, String, char, bool, int, long 등으로 만들 수 있음

```

void testMethod() {
    // TODO ; your method or function
}

```

### 오버로딩

1. 같은 메서드
2. 파라미터 수가 다름 or
3. 파라미터 타입이 다름

(단, 리턴 값만 다를 때는 사용 불가 - 에러)

```
void testMethod() {

}

void testMethod(int a) {

}

void testMethod(String s) {

}

```
