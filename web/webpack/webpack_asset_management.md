## Webpack Management

### npm install loader
```
npm install --save-dev css-loader style-loader file-loader csv-loader xml-loader
```

- css-loader
   ```
   .css를 로딩
   ```
- style-loader
    ```
    .css 파일을 head에 style 태그로 삽입
    ```
- file-loader
    ```
    image, font 등 파일 로딩
    ```
- xml-loader
    ```
    xml 파일 로딩
    ```
- csv-loader
    ```
    csv 파일 로딩
    ```
